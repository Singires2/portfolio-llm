# Data & ML Portfolio

This is a scaffolded portfolio (Next.js + TypeScript + Tailwind) with a demo FastAPI backend in `demo-api/`. Use this as a starting point: customize content, projects, and visuals.

Basic commands:
- Install frontend deps: `npm install`
- Run dev server: `npm run dev`
- Run demo API: `python -m uvicorn demo-api.app:app --reload --port 8000`
